Surgimento da internet

(voltar)

A internet surgiu lá pelos anos 60, no meio da Guerra Fria, como um projeto militar dos EUA chamado ARPANET. A ideia era criar uma rede de comunicação que funcionasse mesmo se parte dela fosse destruída. Em 1969, rolou a primeira conexão entre dois computadores. Nos anos 80, a coisa começou a crescer, saindo do meio militar e chegando nas universidades. Já nos anos 90, com a criação da World Wide Web (WWW) por Tim Berners-Lee, a internet virou o que a gente conhece hoje: um espaço cheio de sites, informações e memes. Aí foi só expansão dela e nunca mais parou.

Html

(voltar)

HTML, que significa HyperText Markup Language (ou Linguagem de Marcação de Hipertexto), é a base de qualquer site que existe na internet. Ele serve para estruturar todo o conteúdo que aparece em uma página da web, como textos, imagens, vídeos, botões e links. Com o HTML, você indica ao navegador o que deve ser exibido e qual a função de cada parte do conteúdo. Por exemplo, você pode usar uma tag para mostrar um título, outra para colocar um parágrafo e outra para inserir uma imagem. Apesar de ser chamado de linguagem, o HTML não é uma linguagem de programação — ele não faz cálculos, nem executa ações complexas. Ele apenas organiza e apresenta as informações de forma que o navegador entenda. Para mudar a aparência dos elementos, como cores, tamanhos e fontes, usamos o CSS. E para deixar a página interativa, com animações ou respostas ao clique do usuário, usamos o JavaScript. Ou seja, o HTML é como o esqueleto de um site: tudo começa por ele.

Tags

(voltar)

As tags são os principais elementos usados no HTML para construir páginas da web. Elas funcionam como marcadores que dizem ao navegador qual é a função de cada parte do conteúdo. Por exemplo, se você quer mostrar um título, uma imagem ou um parágrafo, você usa diferentes tags para isso. Cada tag tem um nome específico e normalmente vem em pares: uma de abertura e outra de fechamento. A tag de abertura indica onde o conteúdo começa e a de fechamento mostra onde ele termina. O conteúdo fica entre essas duas partes. Existem também algumas tags que não precisam de fechamento, como a de imagem. No geral, as tags são fundamentais porque sem elas o navegador não saberia como organizar nem exibir corretamente as informações em uma página.
